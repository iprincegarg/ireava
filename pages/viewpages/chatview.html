Chat System

Ireava chat system module allows a seamless engagement with staff, parents and students.

Internal Chatting Module


  • One to one chat between staff, parents and students.
  • Create group of users for group chat and discussion.
  • Send attachment with the one-to-one or group chat.
  • Send voice note with the one-to-one or group chat.
  • Get notification about the chats.
  • One-way or two-way chatting.
  • Control whether student can initiate chat with the staff.
  • Admin can monitor every one to one and group chat.
  • Chat privacy from the other staff and school members.
  • Send any message reply, mention any user in the group chat.
  • React on any group message.
  • School staff has full control of the group and one to one chat.
