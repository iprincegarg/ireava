Lesson Planning

Ireava provides complete lesson planning solution to manage topic wise content, syllabus status and so on.

Lesson Planning Module Features


  • Manage Subject Chapters, topics.
  • Provide Content in any format to the students.
  • Provide topic wise content to the students.
  • Manage syllabus status done in class.
  • Import, export lesson plan using excel that will save the time.
  • Schedule date when to show lesson plan content to the students.
  • Track syllabus status of every subject.
