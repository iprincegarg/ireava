Online Exam Management

Ireava provides you a complete online exam management. Conduct online exams, get record history, update result and provide solution file.

Online Exam Module Features


  • Conduct online examination for the school students.
  • Create and plan exams that will held in the future.
  • Create a question bank for the exams.
  • Create subjective, objective, multi select, integer, fill up, match box, true false type questions.
  • Students can add bookmark in the question.
  • Shuffle questions subject wise to prevent from cheating.
  • Add multiple subjects in a single exam.
  • Get exam result subject wise marking if multiple subjects added in single exam.
  • Students can get summary of questions status like: unattempted questions, attempted questions, bookmarked questions and not visited questions.
  • Stop students for switching tabs during exam so that it make sure students will not search the answer during exams.
  • Assign attempts for the exam, date and time for exam start-end and duration of exam.
  • Assign students option so that make sure only active students can give exam.
  • Add solution or any information regarding the question so that after result students can review it.
  • Get complete exam marks and status records.
  • Publish result and solutions for the exam.
