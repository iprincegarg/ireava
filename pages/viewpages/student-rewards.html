student Reward Points

Ireava provides a student reward points module in which teachers can provide reward points to the students so that competitive environment will be created between them.

Student Reward Points module Features


  • School can create reward list.
  • Teachers can provide reward points to the students.
  • Students/Parents can check the total reward points earned.
  • School can clear reward points and can again start from zero anytime.

Student Reward Points Reports type

Earned Reward Points Report
