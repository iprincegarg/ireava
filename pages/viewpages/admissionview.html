Admission Management

Ireava provides you a complete student admission management. Add new students with custom and standard fields, get online admissions from school website.

Student Admission Module Features


  • Enrol student with 40+ standard fields.
  • Add custom fields in student form to make it flexible.
  • Import students using csv file at the first time for bulk entries and save a lot of time.
  • Auto Assign fees to the new or old students.
  • Auto generate admission number with custom settings.
  • Manage online admissions from school website or from anywhere.
  • Optional admission fee online payment option for students who enrolled using online admission form.
  • Auto admission done if payment gateway is enabled means everything will be done by system itself.
  • Add custom fields in the online admission form.
  • Enrol students with placement test marks criteria.
  • Get students admission report.
