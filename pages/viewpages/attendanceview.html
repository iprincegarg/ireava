Attendance Management

Ireava provides the multiple way to mark student's and teacher's attendance. Either you can mark manually or by using RFID to automate attendance system.

Student Attendance Module Features


  • Assigned Class Teacher can mark attendance for a class using a mobile application.
  • Admin can mark attendance for any class from admin panel.
  • Super Admin can edit marked attendance for any class from super admin panel.
  • Add attendance points for attendance type to calculate student attendance performance.
  • Mark Students attendance using the barcode attendance application and also mark manual attendance using the application.
  • Automate attendance system for student's using RFID Card and Reader.
  • Scan students barcode using the Ireava barcode appilcation for the attendance marking.
  • Automatic holiday marked for the classes as added in the school holiday academic calendar.
  • Automatic leave applied in attendance as accepted by the admin or teacher.
  • Track check In and check Out record in student's attendance.
  • Access attendance report of a class datewise and monthwise.
  • Access Individual student attendance report.
  • Auto Send email, SMS, application notification to parents on student's check In and check Out time.
  • Get the daily attendance report of a whole school and a class.
  • Get continuously absent student's name on dashboard.


Staff Attendance Module Features


  • Admin can mark attendance for any staff from admin panel.
  • Super Admin can edit marked attendance for any staff from super admin panel.
  • Automate attendance system for staff using RFID Card and Reader.
  • Automatic leave applied in attendance as accepted by the admin or teacher.
  • Salary calculation as per the staff monthly attendance.
  • Track check In and check Out record in staff attendance.
  • Access attendance report of a staff datewise and monthwise.
  • Access Individual staff attendance report.
  • Auto Send email, SMS, application notification to staff on check In and check Out time.
  • Get the daily attendance report of a whole staff.
  • Get absent staff name on dashboard to manage substitute staff.
