Library/E-library Management

Manage library books details, books availability, issue and return details.

Library/E-library Module Features


  • Manage books, journals, newspaper, magazines, articels and others with book shelf.
  • Manage cover photo and e-copy for all the library materials.
  • Find availability of materials with the shelf number.
  • Enable/disable book view or download option.
  • Import materials at bulk using csv file.
  • Manage books details, types and number of books.
  • Manage issue and return of books.
  • Book fine and colection report.
  • Issue book manually or by using bar code.
