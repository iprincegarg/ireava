Homework Management

Ireava allows you to provide homework module to the students.

Homework Module Features


  • Send homework to the students and mark complete to track progress report.
  • Get the students report who done the homework.
  • Notify the students for the Re-Submit if done wrong.
  • Teachers can provide review on the homework.
  • Students can upload images and file in the homework revert option.

Homework Reports type

Homework Complete Report

Homework Progress Report

Homework Due Report
