Tasks and Reminders Manager

Ireava Task and Reminders management system helps you to manage all the tasks of personal, other staffs and reminders of the renewal and expire dates.

Tasks and Reminders Manager


  • Manage tasks and get future reminders about them.
  • Set tasks for your staff and get status of completion.
  • Mark task completion or not status.
  • Students can add their own tasks too.
  • Set reminders for the docs renewal, expire date and so on.
  • Set reminders for personal tasks or work.
