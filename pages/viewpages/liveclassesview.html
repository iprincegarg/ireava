Live Class System

Ireava provides live classes integration to the schools so that they can teach students online and also can provide recorded video lectures for study.

Live Class Module


  • Take live classes within the application.
  • Any teacher can create live class in less than 30 seconds.
  • Super admin can create live class for any teacher.
  • Super admin and admin can attend any ongoing live class.
  • Send real time on click push notification to students for ongoing live class.
  • Zoom live class using separate zoom app and school app.
  • Take live classes using google meet license.
  • Take live classes using webex license.
  • Take live classes using YouTube live.
  • Schedule live classes for a month.
  • Get ongoing, upcoming and completed classes details.
  • Share recorded videos after completion of classes.
  • Automate school attendance with the live class attendance.
  • Students present list in the live class.
  • Any staff can create staff meeting with staff selection option.
  • Send real time on click push notifications for staff meeting.
